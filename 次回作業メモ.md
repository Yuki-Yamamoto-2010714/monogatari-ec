## 💡 トラブルシューティング

### データが表示されない場合
1. ブラウザのハードリフレッシュ（Cmd + Shift + R）
2. Sanityスタジオでドキュメントが「Published」か確認
3. CORS設定を確認（Sanity管理画面）

### ビルドエラーの場合
```bash
rm -rf .next
npm run build
```

### Vercelデプロイエラーの場合
- Vercelの環境変数が正しいか確認
- ローカルで `npm run build` が成功するか確認
